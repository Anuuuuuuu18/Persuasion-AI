# Persuasion-AI
An AI system that analyzes text, detects persuasion strategies, and generates optimized persuasive responses.
What Persuasion-AI Does
Given any text (email, ad, debate, sales pitch, social media post), it will:
1)Detect persuasion techniques
2)Score persuasion strength
3)Explain why it works
4)Rewrite it to be more persuasive
5)Prevent manipulation
